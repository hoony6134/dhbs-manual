파일 업로드 방법
메인 화면
상단 메뉴바에서 파일 메뉴 클릭
모든 파일 > drive 선택
즐겨찾기 (별표) 표시된 drive를 클릭
Drag & Drop (파일 끌어서 서버 화면에서 놓기) 또는 상단 + 버튼 클릭하여 업로드
파일에서 우클릭 후 작업 가능
설문지 작성 기능
신입 방송부원 선발, 설문조사, 초상권 동의 등등 다양한 용도로 사용할 수 있습니다
상단 메뉴바에서 ‘Forms’ 클릭
새로운 형식 클릭 후 Google 설문지와 동일한 방법으로 설문 작성
설문지 공유 방법
우측 상단의 공유 버튼 클릭 후 사이드바의 ‘링크 공유’에 있는 + 버튼 클릭 (방송부 외부도 접근 가능한 링크)
방송부원만 봐야 한다면 아래 ‘내부 링크’ 클릭!
채팅 기능 사용 방법 (토크)
번역이 좀 이상하지만..
채팅방 생성
개인 계정으로 가입할 경우 사용하기 편한 기능입니다.
‘+’ 버튼 클릭 후 새로운 대화 그룹 생성 가능
방송부 외부에서 채팅을 보거나 익명으로 의견을 받을 수 있도록 하려면 게스트가 링크를 통해 참여하도록 허용 옵션을 켜면 됩니다! (암호 설정도 가능)
이후 원하는 참여자를 추가할 수 있습니다
그냥 일반적인 채팅 앱(Instagram DM, Discord, 카카오톡 등)처럼 채팅을 입력하거나 파일, 이미지를 첨부하고, 음성메시지나 이모티콘 등을 사용할 수도 있습니다.
설정 버튼을 누르면 고급 설정을 변경하거나 대화 링크를 복사할 수 있습니다.
‘통화 시작’ 버튼을 누르면 줌과 동일하게 회의를 할 수 있으며, 화면 공유나 마이크/비디오 사용이 가능합니다.
(다만 많은 메모리를 잡아먹어 서버 속도를 저하시킬 수 있으니 사용에 유의해 주세요! — 테스트하고 서버 CPU 47도까지 올랐습니다..)
공지 채팅방
방송부에서 외부로 공지해야 할 내용이 있을 때 사용하기 위해 공지 채팅방을 만들었습니다.

공지는 관리자 계정(서버 관리용으로 사용되는 루트 계정)과 방송부 계정에서만 작성할 수 있으며, 로그인하지 않은 게스트 사용자는 채팅을 작성할 수 없습니다!
